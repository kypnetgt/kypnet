    Detailed Features List

    Below is the detailed list of feature of module Odoo Facebook Catalog Integration

    Allows Odoo and Facebook Catalog Manager Integration

  • Sync and store product data from Odoo to Facebook Catalog Manager.
  • The auto-created XML feeds allow you to upload the product to your FB shop catalog.
  • You can update the FB shop catalog manually or automatically.

    • Control Product Feed Sync from Odoo

  • Set to update the product feed manually or automatically to the FB catalog.
  • With the manual update settings, upload files to FB Catalog Manager.
  • Automatic settings will fetch files automatically as per set interval.
  • The generated token will let you access the feed in Odoo automatically.

    • Select the Warehouse to Import Product Stock to FB Catalog

  • You can choose a warehouse for each product catalog configuration in Odoo.
  • The products will be sent from this warehouse to the Facebook Catalog Manager.
  • Manage the product feed stock from the selected warehouse.

    • Enables you to Map Facebook and Odoo Product Fields

  • With this module, you will get a default configuration of field mappings already saved in it.
  • Also, you can make more field mappings configurations in Odoo.
  • These help match Odoo and FB product fields to save data accurately on Facebook.

    • Create Custom Product Feeds

  • Select which product to include in the product feed of a configuration.
  • Choose Products through the domain or product categories or manually in Odoo.

    • Odoo Facebook Catalog Integration is Multi-Website Compatible

  • The module supports multiple websites.
  • Create separate Feed configurations for each of your Odoo Websites.

    • The Module is Multi-Lingual Compatible

  • Choose the Language in which you want to sync the product feed to the FB catalog from Odoo.
  • The customers can view the catalog in the selected language.

    • Create Multiple Product Feed Configurations

  • Create different product feeds for your multiple FB shop catalogs.
  • Every configuration is independent of each other.
  • You can update the product feed from each to a FB catalog.

    • Sync Google Category with Odoo Website Categories

  • Allot Google Categories to Odoo Website Products to categorize them on Facebook.
  • Import these to Odoo from the static folder in the downloaded zip. (For more information, refer to the User Guide).
  • With the module, you will provide a list of Google support Categories.

    • Permits you to Sync Product Multi-Images

  • The module allows you to sync multiple product images to the FB Shop Catalog.
  • On the FB shop, these product images will appear as a multi-image carousel.

    • Added Facebook Attributes at Product Level

  • Now, add Facebook product Attributes directly in the product form.
  • Also, you can add Google categories, Brand, MPN no., and GTIN to the products.

    • Added option to Sync Product Variants

  • Allows you to sync Odoo Product Variants to the FB Catalog Manager.
  • The admin can display the variants on the respective FB shop in the product configurator.
